Kubernetes Health and Security Score

k8sgrade

A Bash tool that inspects your cluster namespace, scores operational and security posture, and gives practical fixes.

Open Repository

Quick Start

./k8sgrade.sh -c your-context -n your-namespace

Run in interactive mode if you want guided context and namespace selection.

./k8sgrade.sh

What It Checks

  • Node and pod health
  • Workload safety controls
  • Network and disruption protections
  • Service account and RBAC risk
  • Trivy vulnerability signals

Prerequisites

  • bash
  • kubectl
  • Kubernetes context access
  • Valid kubeconfig

Optional: python3 and trivy.

Exports

  • CSV summary report
  • Score breakdown rows
  • Recommendations list
./k8sgrade.sh -c ctx -n ns --export-csv ./k8sgrade-report.csv

How It Works

  1. Validates prerequisites and input options.
  2. Selects context and namespace.
  3. Collects health and security signals.
  4. Applies score deductions from 100.
  5. Prints report, grade, and optional CSV export.
